# Задача 2. Доставка заказов

Студенческая команда планирует создать стартап службы доставки для обработки заказов от магазинов и предприятий питания и оперативной передачи их курьерам.

Система должна собирать заказы в едином формате: на первом этапе оператор вручную вносит данные о заказе, месте комплектации, адресе доставки и желаемом сроке доставки. Для курьеров разрабатывается мобильное приложение.

Курьер может просматривать свободные заказы, выбирать и бронировать заказ, забирать его в точке выдачи, доставлять клиенту и обновлять статус выполнения в системе.

Диспетчер контролирует работу курьеров и при необходимости переназначает заказы. Данные о заказах и выполненных доставках передаются в бухгалтерскую систему для расчетов с поставщиками и курьерами. Начисленная оплата отображается в личном кабинете курьера.

Администратор регистрирует курьеров и назначает пользователям права доступа.
