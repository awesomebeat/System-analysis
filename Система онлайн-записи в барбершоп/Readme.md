# Задача 1. Запись на стрижку

Руководство сети барбершопов планирует внедрить систему онлайн-записи для расширения клиентской базы, снижения ручной нагрузки на сотрудников и автоматизации информирования клиентов.

Система позволяет зарегистрированным и незарегистрированным посетителям выбирать тип услуги, конкретную услугу, мастера и свободное время записи. Клиенты получают автоматические напоминания через выбранный канал связи: Telegram, WhatsApp, VK или СМС. После оказания услуги система предлагает оставить оценку и отзыв.

Менеджер ведет расписание мастеров, указывает доступные услуги, корректирует записи, связывается с клиентами вручную, отмечает выполнение услуги, фиксирует оплату и передает данные в бухгалтерию. Также менеджер просматривает отчеты и отзывы клиентов.

Мастер может просматривать свое расписание, записи на услуги и отзывы клиентов.
