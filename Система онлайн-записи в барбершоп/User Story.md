# User Stories: менеджер

## US-mng-01. Поддерживать актуальность расписания

**User Story:**  
Я, как менеджер точки барбершопа, хочу иметь возможность редактировать расписание мастеров в системе, чтобы обеспечить актуальность расписания.

| Поле | Описание |
|---|---|
| ID | US-mng-01 |
| Заголовок | Поддерживать актуальность расписания |
| Роль пользователя | Менеджер |
| Ценность | Предотвращение накладок и ошибок в расписании, благодаря чему повышается удовлетворенность клиентов |
| Требуемые действия | Открыть расписание, изменить в записи время / мастера / услугу, сохранить изменения |

### Критерии приемки

| Раздел | Описание |
|---|---|
| Предусловие | Менеджер авторизован в системе |
| Предусловие | В системе зарегистрированы мастера, указан их рабочий график |
| Предусловие | В системе уже есть записи клиентов |
| Действие 1 | Менеджер открывает раздел «Расписание» |
| Действие 2 | Менеджер выбирает определенную запись |
| Действие 3 | Менеджер корректирует информацию в записи |
| Действие 4 | Менеджер сохраняет изменения |
| Результат | Менеджер видит актуальное расписание, это позволяет избежать ошибок при добавлении новых записей |

## US-mng-02. Просматривать и контролировать отзывы

**User Story:**  
Я, как менеджер точки барбершопа, хочу иметь доступ к отзывам клиентов и возможность их систематизации.

| Поле | Описание |
|---|---|
| ID | US-mng-02 |
| Заголовок | Просматривать и контролировать отзывы |
| Роль пользователя | Менеджер |
| Ценность | Повышение качества обслуживания клиентов |
| Требуемые действия | Открыть отзывы клиентов, отфильтровать по мастеру / оценке |

### Тестовый сценарий

| Раздел | Описание |
|---|---|
| Предусловие | Менеджер авторизован в системе |
| Предусловие | В системе уже есть отзывы, оставленные клиентами |
| Действие 1 | Менеджер заходит в раздел «Отзывы» |
| Действие 2 | Система отображает список всех отзывов |
| Действие 3 | Менеджер выбирает период, за который необходимо просмотреть отзывы |
| Действие 4 | Менеджер сортирует отзывы по рейтингу / мастеру |
| Ожидаемый результат | Отображается список отсортированных отзывов за выбранный период |

# User Stories: клиент

## US-clt-01. Записаться на услугу онлайн

**User Story:**  
Я, как клиент барбершопа, хочу записаться на услугу онлайн, выбрав мастера и время, чтобы не тратить время на звонки и ожидание.

| Поле | Описание |
|---|---|
| ID | US-clt-01 |
| Заголовок | Записаться на услугу онлайн |
| Роль пользователя | Клиент |
| Ценность | Экономия времени и удобство |
| Требуемые действия | Выбрать услугу, мастера, время и подтвердить запись |

### Критерии приемки

| Раздел | Описание |
|---|---|
| Предусловие | У клиента есть доступ к системе |
| Предусловие | В системе есть список услуг и мастеров |
| Предусловие | Система отображает свободные слоты для записи |
| Действие 1 | Клиент выбирает тип и вид услуги |
| Действие 2 | Клиент выбирает мастера |
| Действие 3 | Клиент выбирает свободный временной слот |
| Действие 4 | Клиент подтверждает запись |
| Результат | Клиент получит необходимую услугу в удобное время |

---

## US-clt-02. Оставить отзыв

**User Story:**  
Я, как клиент барбершопа, хочу иметь возможность оставить отзыв в системе после визита, чтобы поделиться впечатлениями и помочь другим клиентам с выбором.

| Поле | Описание |
|---|---|
| ID | US-clt-02 |
| Заголовок | Оставить отзыв |
| Роль пользователя | Клиент |
| Ценность | Влияние на качество сервиса |
| Требуемые действия | Получить уведомление от системы о возможности оставить отзыв, заполнить форму отзыва, отправить отзыв |

### Тестовый сценарий

| Раздел | Описание |
|---|---|
| Предусловие | Статус услуги в системе изменен на «Выполнена» |
| Предусловие | Система отправила клиенту предложение оставить отзыв |
| Предусловие | У клиента есть доступ к системе |
| Действие 1 | Клиент открывает ссылку / уведомление с предложением оставить отзыв |
| Действие 2 | Система отображает форму отзыва |
| Действие 3 | Клиент заполняет форму |
| Действие 4 | Клиент отправляет форму |
| Ожидаемый результат | Отзыв сохранен в системе и виден менеджеру и мастеру |
